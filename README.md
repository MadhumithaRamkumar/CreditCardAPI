This project implements an API for credit card number validation using the Luhn algorithm. The API can be tested through the Swagger interface available on the output screen.
It accepts a credit card number as input (in string format) and returns a response indicating the validation status and related details of the card number.

**FrameWorks**
1.  .Net 9 Framework - C# .NET Framework

**   SETUP INSTRUCTION**

1.  Open Visual Studio.
2.  Click on "Clone on repository".
3.  Paste the Git repo URL in the Repository location field.
4.  Choose a local path where the repo will be saved.
5.  Select Clone.
6.  Also clone the CreditCardAPI.test to the local path using the same above steps to run the unit test.
